# Finance Stock Tracker - MERN Stack Application

A full-stack web application for tracking stock prices and news using the **Finnhub API**. Built with **React**, **Node.js**, **Express**, and **MongoDB**.

## 🚀 Features

- **Real-time Stock Prices**: Search and view current stock prices
- **Company Information**: Get detailed company profiles and market data
- **Latest News**: Read the latest news about stocks
- **Favorites**: Save your favorite stocks for quick access
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Live Market Data**: Powered by Finnhub API

## 📦 Tech Stack

### Frontend
- React 18
- React Router DOM
- Axios
- Recharts (for charts)
- React Icons
- CSS3

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Axios
- CORS
- Morgan

### APIs
- Finnhub API (Stock Data)
